.. Getting Started

===============
Getting Started
==============

This assumes you already have the package installed.

.. code-block:: none

  from hypyxel import utils, hypixel
  
  utils.set_api_key("your_key")
  hypixel.player("username")
  
 For a complete list of endpoints, go either `here <https://hypyxel.rtfd.io/en/latest/skyblock_endpoints.html>`__ for the list of skyblock endpoints, or `here <https://hypyxel.rtfd.io/en/latest/hypixel_endpoints.html>`__ for a general list of endpoints.
